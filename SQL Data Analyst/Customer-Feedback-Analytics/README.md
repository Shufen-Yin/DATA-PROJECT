
# Customer Feedback Analytics (SQL + Python)

**Role:** Data Analyst / SQL Analyst  
**Focus:** Database Design • ETL • Analytics

---

## Project Overview

This project demonstrates a complete analytics workflow using:

- **SQL** for database schema and relational design
- **Python** for ETL, data validation, and analysis
- Multi-source customer feedback in **CSV, JSON, and XML** formats

The objective is to convert raw customer feedback into structured data and extract actionable insights for business decision-making.

---

## Tech Stack

- SQL (schema design)
- Python (csv, json, xml, sqlite3)
- SQLite (lightweight analytics database)

# Data Analysis & Data Science Portfolio

This portfolio showcases my skills in **data analysis, machine learning, and AI**, using **Python, SQL, Tableau, and PowerBI**.  
It contains end-to-end projects demonstrating **ETL, EDA, feature engineering, ML modeling, imbalance handling, hyperparameter tuning, NLP, and visualization**.

---

## Skills Demonstrated

- **ETL & Data Wrangling**: Cleaning, feature engineering, scaling, one-hot encoding  
- **Exploratory Data Analysis (EDA)**: Statistics, distributions, correlations, visualizations  
- **Machine Learning & AI**: Regression, Classification, Random Forest, Time Series, Clustering, Imbalanced Data Handling (SMOTE, Class Weighting), Hyperparameter Tuning, Dimensionality Reduction  
- **NLP & Text Analytics**: Tokenization, vectorization, sentiment analysis  
- **Data Visualization**: Matplotlib, Seaborn, Tableau, PowerBI  
- **Model Evaluation & Monitoring**: Accuracy, Precision, Recall, F1, ROC-AUC, confusion matrix  
- **Deployment Concepts**: REST API, batch scoring, monitoring metrics
